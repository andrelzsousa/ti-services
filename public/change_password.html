Requisitos da Senha:

  • Pelo menos 6 caracteres
  • Pelo menos 1 número (0-9)
  • Pelo menos 1 letra maiúscula (A-Z)
  • Pelo menos 1 caractere especial permitido: @ # $ % & ! ? / + = .
  • Caracteres NÃO permitidos: { } [ ] < > ;
